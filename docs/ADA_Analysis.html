Typical Attendance During Consecutive Weeks of Dosage

This analysis looks at the typical district present days during the consecutive weeks served. Assuming each week is a whole school week, the ADA % is calculated by dividing present days with number of consecutive weeks served times 5. If a student has multiple similar consecutive weeks served, only the ADA on the latest consecutive week is included in the analysis
